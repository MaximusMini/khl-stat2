Алгоритм формирования постера встречи 2-команд

	1. 	Выбираются из выпадающего списка две команды
			+	дата матча
			+	время начала матча
			+	название ледовой площадки
			+	город проведения матча
	
	2.	Из БД получаются необходимые показатели 2-команд
			- для этого создается модель - в которой формируется соотвествующий класс
			- экземпляр класса имеет необходимые методы для получения показателей из БД
			- на входе экземпляр класса получает id команды
			- после описания класса (в моделе) происходит вызов соотвествующих методов для получения данных команды
			- в конце работы модели получается объект (команды) с необходимыми данными
	
	3.	На основании полученных показателей формируется итоговый постер
			
	
	4.	Постер сохраняется на диск
